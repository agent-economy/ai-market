# 한국형 웹디자인 레퍼런스 가이드

## 🎯 핵심 인사이트: 한국 vs 글로벌 디자인

### 현재 우리 디자인 (글로벌/테크 트렌드)
- 다크 배경 + 글래스모피즘
- 보라/파랑 네온 그라데이션
- 기하학적 패턴
- "AI가 만든" 느낌

### 한국 타겟이 원하는 디자인
- 밝은 배경 + 여백 미학
- 베이지/크림/우드톤 (따뜻함)
- 감성 폰트 + 손글씨 느낌
- "사람이 정성껏 만든" 느낌

## 🎨 한국형 컬러 팔레트

### A. 카페/베이커리 (따뜻한)
```css
--warm-bg: #FDF8F3;        /* 크림 배경 */

--warm-text: #3D3D3D;      /* 차콜 */
--warm-muted: #9B9B9B;     /* 그레이 */
```

### B. 인스타 감성 (깔끔한)
```css
--insta-bg: #FFFFFF;       /* 퓨어 화이트 */

--insta-accent: #E8E8E8;   /* 라이트 그레이 */
--insta-point: #D4A574;    /* 테라코타 */

```

### C. 뉴트로/레트로 (힙한)
```css
--retro-bg: #F5F0E8;       /* 아이보리 */
```

### D. 모던 미니멀 (세련된)

## 📐 레이아웃 패턴

### 한국 소상공인 사이트 공통점
1. **풀스크린 히어로** — 가게 대표 사진 + 한 줄 카피
2. **여백 많이** — 답답하지 않게, 숨 쉴 공간
3. **아이콘 대신 사진** — 실제 음식/공간 사진이 신뢰감
4. **SNS 연결** — 인스타그램 피드 임베드
5. **카카오 채널 버튼** — 네이버 예약보다 친숙

### 섹션 순서 (추천)
1. 히어로 (가게명 + 사진)
2. 소개 (스토리)
3. 메뉴/서비스
4. 갤러리 (사진 그리드)
5. 위치 + 영업시간
6. 예약/문의 버튼

3. menu-1.html: 그라데이션 → 깔끔한 그리드
